# Python List Comprehensions - Basic to Advanced

This repository contains a simple Python tutorial showing how to use list comprehensions for common tasks.

## Contents

- `PYTHON LIST COMPREHENSIONS - BASIC TO ADVANCED.py` - A script demonstrating:
  - plain list construction vs. list comprehension
  - generating squares and cubes
  - filtering even and odd numbers
  - transforming values with multiplication and uppercasing strings
  - computing string lengths
  - using conditional expressions in list comprehensions
  - filtering positive numbers

## Requirements

- Python 3.x

## How to Run

From the repository folder, run:

```bash
python "PYTHON LIST COMPREHENSIONS - BASIC TO ADVANCED.py"
```

## Purpose

This project is intended as a learning resource for Python beginners who want to understand list comprehensions and see examples of basic to advanced usage patterns.

## Notes

- The script prints each example result to the console.
- The file name contains spaces, so quote it when running from the command line.
